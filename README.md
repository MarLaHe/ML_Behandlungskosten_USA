In diesem Projekt werden Machine Learning Modelle trainiert, um anhand verschiedener Merkmale die zukünftigen Behandlungskosten für Neukunden von Krankenkassen zu schätzen.

Der Datensatz stammt von kaggle und wurde unter dem Titel "Insurance Claim Analysis: Demographic and Health" veröffentlicht.
<br> Zum Datensatz bekannt ist allein, dass es sich um Daten aus den USA handelt.

Die komplette Ausarbeitung des Projektes inklusiv ausführlichen Kommentaren und zusätzlichen Informationen befindet sich im Notebook. Begleitend wurde eine Präsentation erstellt.
